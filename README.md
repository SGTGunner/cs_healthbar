# CS_Healthbar

Installation guide:

1. Download file `git clone https://github.com/SGTGunner/cs_healthbar.git`
2. Drop cs_healthbar in yours resource folder
3. Add `start cs_healthbar` to `server.cfg`

Have fun with editing this :D
